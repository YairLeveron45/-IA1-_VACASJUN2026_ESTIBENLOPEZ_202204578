# Setup del backend

## Pasos sugeridos

1. Crear un entorno virtual.
2. Instalar dependencias con `pip install -r requirements.txt`.
3. Ejecutar la API con `uvicorn app.main:app --reload`.

## Notas

- Asegurate de tener SWI-Prolog instalado.
- Configura las credenciales de Telegram antes de habilitar notificaciones.
